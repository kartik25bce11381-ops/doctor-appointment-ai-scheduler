## Project Overview
This project implements a **Rule-Based Intelligent System** that uses **Forward Chaining** to intelligently schedule doctor appointments. It demonstrates core concepts from Unit 2 of the course, including knowledge representation, production rules, and problem-solving methods in Artificial Intelligence.

The system allows patients to book appointments with available doctors based on speciality and time slots. It checks rules in sequence (Forward Chaining) and provides real-time confirmation or appropriate error messages.

## Features
- Rule-Based Forward Chaining for intelligent decision making
- Support for multiple doctors and specialities
- Real-time availability checking
- Input validation and user-friendly error handling
- Fully Command Line Interface (CLI) based application
